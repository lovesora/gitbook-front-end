# shortcut
* 分屏
```
command + d                     --------- 水平分屏
command + shift + d             --------- 垂直分屏
command + [                     --------- 逆时针顺序切换
command + ]                     --------- 顺时针顺序切换
option + command + arrow        --------- 按照箭头方向切换
```

* tab
```
command + t                     --------- 创建一个tab
command + {/}                   --------- 切换tab
command + left/rigth arrow      --------- 切换tab
command + option + e            --------- 全屏显示所有tab
```

* 控制
```
command + o                     --------- 打开配置选择

command + shift + h             --------- 打开历史粘贴记录
command + ;                     --------- 弹出自动补齐窗口

command + /                     --------- 高亮鼠标位置

tab                             --------- 自动补全
tab + tab                       --------- 显示目录下所有文件
```
